## Сессия #3 — 29 мая 2026
**Задача:** Заменить D3.js график на Lightweight Charts (TradingView)

**Что сделано:**
- Убран D3.js (CDN), подключён lightweight-charts v4.1.1
- График теперь рендерится на Canvas через библиотеку TradingView — выше производительность, скролл, зум, встроенный кроссхеир
- Все индикаторы (BB, SMA, RSI) переведены на line-series lightweight-charts с отдельным priceScale для RSI
- Инструменты рисования переведены на `createPriceLine()` — горизонтальные линии и линейки с % метками
- Клики по графику обрабатываются через `chart.subscribeClick()`
- Кроссхеир — через `chart.subscribeCrosshairMove()`, OHLC данные в тулбаре
- Данные свечей конвертируются в Unix-секунды для совместимости с lightweight-charts
- WebSocket обновления свечей через `candleSeries.update()`, индикаторы пересчитываются на лету
- При смене монеты/таймфрейма график переинициализируется, старые priceLine очищаются
- Ресайз графика встроенный (lightweight-charts сам отслеживает размер контейнера)

**Файлы изменены:**
- `/root/scrineer/index.html` — замена D3.js на lightweight-charts, переработка всей графической части

## Сессия #4 — 30 мая 2026
**Задача:** Индикаторы выкл по умолчанию, полная точность цены, магнит, луч вправо, фракталы

**Что сделано:**
- Индикаторы (BB, SMA, RSI) выключены по умолчанию — включаются только по желанию через меню
- Правая шкала цены — полная точность без округлений (`priceFormat: custom`, все цифры как с биржи)
- Магнит работает: при активации курсор/клик притягивается к ближайшему OHLC-значению свечи
- Луч рисуется от точки клика только вправо (через line-series, не через PriceLine)
- Автопоиск фракталов (Williams, шаг 3): экстремумы ищутся автоматически, рисуются горизонтальными уровнями
  - Красные линии — верхние фракталы (high)
  - Зелёные линии — нижние фракталы (low)
  - Фракталы пересчитываются при каждой новой свече, смене таймфрейма/монеты
- Ластик очищает всё: линии, лучи, фракталы

**Файлы изменены:**
- `/root/scrineer/index.html` — множественные правки графического движка
- `/root/scrineer/SESSIONS.md` — запись сессии #4
